# Scantastic [1.0.0]
A simple web GUI for hosting audio snippets intended for use with SDR scanner utilities.
<br><br>
REBROADCASTING CERTAIN TRANSMISSIONS, SUCH AS POLICE TRAFFIC, CAN BE ILLEGAL IN SOME REGIONS! Do not use this tool unless you're aware of your legal responsibilities with regards to radio scanning.
<br><br>
